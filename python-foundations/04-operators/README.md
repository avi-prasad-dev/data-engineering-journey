# Topic 4: Operators
Operators allow us to perform calculations and comparisons.

### Examples:
    10 + 5
    20 - 5
    10 * 2
    20 / 4

Without operators, software couldn't:
- Calculate salaries
- Compute taxes
- Process orders
- Compare passwords
- Validate user ages

## Types of Operators
## Arithmetic Operators
    +
    -
    *
    /
    %
    **
    //

## Comparison Operators
    ==
    !=
    >
    <
    >=
    <=

## Logical Operators
    and
    or
    not
