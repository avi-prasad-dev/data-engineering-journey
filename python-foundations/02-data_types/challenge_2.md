# Coding Challenge
## Goal

### Create a program that stores the following information about yourself.

### Inputs
    - Full Name
    - Age
    - Monthly Salary (realistic or dummy value)
    - City
    - Are you learning Python? (True/False)
### Output

Print:

    Name:
    Age:
    Salary:
    City:
    Learning Python:

Then print the data type of every variable using type().

## Difficulty

Easy

## Expected Concepts
    - Variables
    - Data Types
    - Type Hints
    - type()
